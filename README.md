## 🚀 Sobre mim

Desenvolvedor com mais de **3 anos de experiência** no desenvolvimento de software, especializado em aplicações web utilizando **JavaScript** e **PHP**.

Atuo tanto no **front-end** quanto no **back-end**, desenvolvendo e evoluindo sistemas web com tecnologias como **Node.js**, **React**, **Next.js**, **React Native** e **Laravel**.

Possuo experiência completa no ciclo de desenvolvimento, desde o levantamento de requisitos junto ao cliente e a concepção da solução, passando pela criação de protótipos, implementação de funcionalidades e realização de testes, até a entrega de aplicações robustas, escaláveis e alinhadas às necessidades do negócio.
